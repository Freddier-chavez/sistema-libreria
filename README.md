# Sistema-Libreria
¡Hola!, Esta esta es un sitio web libreria desarrollada con python y django + CRUD + MySQL. Puedes agregar, editar, consultar y eliminarla la lista de libros de la bibliteca.
Espero sea apoyo para la comunidad.
